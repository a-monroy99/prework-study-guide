HTML

  • HTML is one of the three cornerstones and is used to create the structure of a webpage.
  • HTML is used up of elements that provide content and the basic framework for all pages on the web, which can make it easy for the a user to navigate the webpage.
  • Elements is a type of content that adds content to the pages. Elements: Text, Image, Header, List.
  • HTML is usually composed of opeing and closed tags "< >".
  • The head element contains the informatioin, or metadata, that helps the browser to render the page correctly
  • HTML attributes defines addittional information about an element
  • The body element represents the visible content shown to the user.
