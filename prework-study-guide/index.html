HTML

  • HTML is one of the three cornerstones and is used to create the structure of a webpage.
  • HTML is used up of elements that provide content and the basic framework for all pages on the web, which can make it easy for the a user to navigate the webpage.
  • Elements is a type of content that adds content to the pages. Elements: Text, Image, Header, List.
  • HTML is usually composed of opeing and closed tags "< >".
  • The head element contains the informatioin, or metadata, that helps the browser to render the page correctly
  • HTML attributes defines addittional information about an element
  • The body element represents the visible content shown to the user.

CSS

  • Casscading Style Sheet - Defines how the webpage is presented to users i.e colors, aesthetics, visual layout, and organization.
  • Inline CSS is used to style an element to a specific HTML tag
  • Internal CSS is used to add a unique style to the webapge in the head section of the HTML file.
  • External CSS is used to change the entire website by changing just one file.
  • By using a class attribute we can assign a class selector and apply it to specific element we choose.

Git

  • Git is version control software, that helps you manage and keep track of each version of your project
  • Git and github are different
  • git checkout -b [branch name] is a way to create and switch to a new branch
  • You can use git revert commnd to revert to the correct version afer the SHA has been selected
  • Git commands always start with [git]
  • Github is cloud-based service. It allows collaboration with others. This hosting service keeps track of our projects inside repositories that are stores remotely.
